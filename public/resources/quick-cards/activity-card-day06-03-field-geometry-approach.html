Day 6 · Session 03
35 min · PM Match-Build-Make
📐 d = √((x₂−x₁)² + (y₂−y₁)²) · Time = Distance × 10 ÷ S · Ball = (4,2) · Goal = (8,2) · justify: start + time + angle
Before Session
  • Board: Ball = (4, 2) · Goal = (8, 2)
  • Coordinate grid posted or projected
  • Each team has their Speed Stat (S) recorded
  • One worksheet per team distributed
Key Distinction
Stat loadout = bot performance parameters (already locked). Launch sequence = startup code controlling initial angle, speed, and timing before ball contact. Students design startup code today — not the loadout.
Produces
  • Committed starting position per team
  • Plotted field: P, B(4,2), G(8,2) with approach path
  • Calculated distance to ball + travel time
  • Written geometric justification
Sequence
0:00Frame it"Your launch sequence controls initial angle, speed, and timing. Where you start determines whether the ball goes to the goal or somewhere else." Do not suggest a starting position.
0:02Commit to startTeams write starting position on worksheet before Phase 2 opens. If they choose (4,2) or (8,2) — redirect without drama.
0:07Plot + drawPlot P, B(4,2), G(8,2). Draw P→B then B→G as continuous arrow. Remind: B→G is the ball's path — not the bot's. Extend P→B line past the ball: does it point toward (8,2)?
0:17Distance calcd = √((x₂−x₁)² + (y₂−y₁)²). "Subtract first, then square." Walk one worked example together first.
0:24Travel timeTime = Distance × 10 ÷ S. Example: d=5, S=8 → 5×10÷8 = 6.25 sec. Then: write 3-part justification.
0:29Share outFastest time? Best angle? Can one position give both? Listen for start + time + direction as one system.
0:35CompleteEvery team has a travel time and a written geometric justification.
Say
Start Q"Where should we start?" → "That's the problem you're solving."
Angle check"Look at your path from P to B. Extend that line past the ball. Where does it point? Is that where the goal is?"
Justification standard"Not accepted: 'We start close so we get there fast.' Accepted: 'We start at (0,2) and approach horizontally. Distance = 4 units, time = 5 sec at S=8. Because we approach along y=2, the ball continues toward (8,2).'"
Stuck on justification"What was your travel time? Say that. Now — where does your path point past the ball? That's the justification."
Watch For
Squaring before subtracting — stress parentheses every time: "Subtract first, then square"
Negative differences confusing students — "(−4)² = 16. Squaring removes the negative."
Flipped x/y coordinates — ask "which number goes across?" before they calculate
Short distance + fast time, no angle check — "Is the angle right though? Check your path."
Hard Stop0:35 — every team has calculated travel time and written 3-part geometric justification
