Setup

Open MakeCode and prep the project

  1. Open MakeCode for micro:bit
    Go to makecode.microbit.org. You do not need to sign in.
  2. Create a new project
    Click New Project, name it velocity-match, and hit Create.
  3. Add the Cutebot Pro extension
    Click the gear icon (top right) → Extensions. Search for cutebot pro and click the tile to add it. New block categories appear in the sidebar when it's installed.
  4. Switch from Blocks to JavaScript
    At the top of the editor, you'll see a toggle with Blocks and JavaScript. Click JavaScript. The block-based workspace swaps for a text editor.
  5. Replace everything with the match template
    Select all the default code (⌘ A on Mac, Ctrl A on Windows/ChromeOS) and delete it. Then paste the code from the block below. Use Copy with my values if you want your stats pre-filled; use Copy template to paste the raw version.
  6. Bring in your launch sequence from your earlier project
    You already wrote a launch sequence in the First Bot Drive project. Pull it into this one:
    1. Open your earlier MakeCode project in a new tab (Home → your projects → the one with your launch sequence).
    2. At the top, click Blocks to see your launch sequence as blocks — this is the quickest way to confirm you're looking at the right function.
    3. Click JavaScript so you can read and copy the text.
    4. Find the lines between launchActive = true and launchActive = false inside launchSequence(). That's your opening-move code. Select just those lines and copy (⌘ C / Ctrl C).
    5. Switch back to velocity-match. In the new code, find the same spot — right where it says // ← YOUR CODE HERE. Paste your lines there.
    Leave the two launchActive = true / launchActive = false lines alone — they're already in place and must not be duplicated.

Run the match

    
Launch and observe

    

      Download the program to your micro:bit. Place the bot on your goal line facing the field.
      Press A to start the match. Press A + B together at any time to
      emergency-stop. The match auto-ends after 3 minutes (180 seconds).
    

    

      Record what you see in your match log: how long did Endurance hold? Did your launch put
