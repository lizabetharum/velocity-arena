Speed Test Program

This program drives the bot straight for 2 seconds at whatever SPEED you've set — so you can see how far it travels at different motorPower values. Press B to cycle through your stat SPEED without re-flashing.

Button Controls
Button A

Run the test

Shows SPEED stat and power bar, then drives straight for 2 seconds and stops.

Button B

Cycle SPEED

Steps SPEED from 1 → 2 → … → 20 → back to 1 without re-flashing. Shows new value on LED.

A + B

Emergency stop

Immediately cuts both motors during a run. Shows X on LED and clears after 500 ms.

How Button B Cycling Works

Button B changes SPEED in memory without touching the flashed program. Each press adds 1 to SPEED. When SPEED hits 20, the next press wraps it back to 1.

1 → 2 → · · · → 20 → 1 → · · ·
ℹ️
The new SPEED only takes effect the next time you press A. Pressing B mid-run changes the value in memory but doesn't affect the current drive — currentPower was already computed when you pressed A.

How to Use This Program
  1. 1 Set SPEED. Then download the program.
  2. 2 Set a SPEED with Button B. Each press adds 1. The scrolling number confirms the current value. Stop when you reach the stat you want to test.
  3. 3 Place the bot on a clear straight run. It drives for 2 seconds then stops. Mark where it starts and where ends up.
  4. 4 Press A to run. Watch the preview (number → bar graph), then it drives. Press A+B any time to stop early.
  5. 5 Compare speeds. Press B to bump SPEED up, reposition the bot at the same start line, press A again. Compare how far it travels.
⚠️
The bot has no sensors active during this test. It drives blind for 2 seconds. Make sure the run path is clear — it will drive off a table or into a wall without stopping.
